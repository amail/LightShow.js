Overlay hide button example

This example demonstrates how you can use the 'hide-button' class in order to create awesome close/hide buttons.

Markup 
<div id="document-overlay-content">
	<h1>Overlay with hide button</h1>
	<p>The button below has the 'hide-button' class attached to it. Click on it!</p>
	<p><input type="button" value="GEEEZ CLICK ME ALREADY" class="hide-button" /></p>
</div>
Code 
$.lightShow({
	content: $("div#document-overlay-content"),
	showOnStartup: true
});
