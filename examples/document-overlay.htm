Document overlay example

This example demonstrates how you can overlay the whole document.

Markup 
<div id="document-overlay-content">
	<h1>This is a full document overlay</h1>
	<a href='#' class='hide-button'>I HAVE SEEN IT</a>
</div>
Code 
$.lightShow({
	content: $("div#document-overlay-content"),
	showOnStartup: true
});
