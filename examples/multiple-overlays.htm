Multiple overlays example

This example demonstrates how you can have multiple overlays overlaying each other on the same page.

Markup 
<div id="element-to-overlay">
	<img src="images/awesome-cat.jpg" width="500" height="375" />
</div>
Code 
$.lightShow({
	overlay: $("div#element-to-overlay"),
	content: "<a href='#' class='hide-button'>Click to close (1st)</a>",
	showOnStartup: true,
	style: {
		opacity: 0.6
	}
});

$.lightShow({
	overlay: $("div#element-to-overlay"),
	content: "<a href='#' class='hide-button'>Click to close (2nd)</a>",
	showOnStartup: true,
	style: {
		opacity: 0.6
	}
});

$.lightShow({
	content: "<a href='#' class='hide-button'>Click to close (3rd)</a>",
	showOnStartup: true
});
Actions
Show | Hide
