Custom overlay effects example

This example demonstrates how easy you set custom show/hide effects.

Markup 
<div id="element-to-overlay">
	<img src="images/awesome-cat.jpg" width="500" height="375" />
</div>

<div id="element-overlay-content">
	<h1>BACHINATOR</h1>
	<a href='#' class='hide-button'>SHOW HIM</a>
</div>

Code

Notice: The custom zoom effect (zoomIn/zoomOut) used here is included from the file 'scripts/effects.jquery.js'.

$.lightShow({
	overlay: $("div#element-to-overlay"),
	content: $("div#element-overlay-content"),
	showOnStartup: true,
	effect: {
		show: function(element, complete_callback){
			element.queue(function(){
				element.zoomIn(500, complete_callback);
			});
		},
		hide: function(element, complete_callback){
			element.queue(function(){
				element.zoomOut(500, complete_callback);
			});
		}
	}
});
