Element hover overlay example

This example demonstrates how you easily can show an overlay when hovering a specific element.

Markup 
<div id="element-to-overlay">
	<img src="images/awesome-cat.jpg" width="500" height="375" />
</div>

<div id="element-overlay-content">
	<h1>You have seen this cat</h1>
</div>
Code 
$.lightShow({
	overlay: $("div#element-to-overlay"),
	content: $("div#element-overlay-content"),
	showOnHover: true
});

You have seen this cat
