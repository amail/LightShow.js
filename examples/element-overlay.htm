Element overlay example

This example demonstrates how you can easily overlay a specific element.

Markup 
<div id="element-to-overlay">
	<img src="images/awesome-cat.jpg" width="500" height="375" />
</div>

<div id="element-overlay-content">
	<h1>Have you seen this cat?</h1>
	<a href='#' class='hide-button'>SEE IT</a>
</div>
Code 
$.lightShow({
	overlay: $("div#element-to-overlay"),
	content: $("div#element-overlay-content"),
	showOnStartup: true
});
Actions
Show | Hide
